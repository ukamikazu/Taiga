Taiga
An attempt at understanding and visually simulating Unorthodox Kitten's Beauty in Infrastructure project

The Taiga framework, nine axioms, Proposition 137.1, and all foundational concepts are the original work of Unorthodox Kitten (Beauty in Infrastructure, [youtube.com/@Unorthodox_Kitten](https://www.youtube.com/@Unorthodox_Kitten)). This repository contains a mathematical formalisation, experimental programme, and interactive simulation built from that framework. MIT license applies to the formalisation and simulation code only, not to the underlying framework.
